# Home Lab

Welcome to the Home Lab documentation. This project walks through building a Windows Server environment from scratch on a home virtualization platform.

## What This Lab Includes

- **Windows Server** running as a virtual machine
- **Active Directory Domain Services (AD DS)** — domain controller, users, and organizational units
- **DNS** — forward and reverse lookup zones for the lab domain
- **DHCP** — dynamic IP assignment for lab clients
